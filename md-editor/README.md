# MD Editor - Markdown エディタ

シンプルな Markdown エディタアプリケーション（独立プロジェクト）。

## 🚀 開発サーバーの起動

```bash
pnpm install
pnpm dev
```

ブラウザで http://localhost:3000 を開いてください。

## 📝 概要

このプロジェクトは Markdown エディタの独立したプロトタイプです。  
メインのコンテンツ管理システムは `editor-home/` ディレクトリにあります。

## 🛠️ 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **UI**: Tailwind CSS
- **パッケージマネージャー**: pnpm
